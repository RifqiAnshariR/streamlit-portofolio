# Assignment: Portfolio Building with Streamlit

## Setup:
1. `git clone https://github.com/RifqiAnshariR/streamlit-portofolio.git`
2. `cd streamlit-portofolio`

## How to run:
1. `py -3.10 -m venv .venv` and activate it `.venv\Scripts\activate`
2. `pip install -r requirements.txt`
3. To run api: `python app.py`
4. To run Streamlit: `streamlit run Home.py`
